Файл cardio-boosting.ipynb содержит

- описание данных
- ссылки на исходные данные
- шаги предобработки
- подбор гиперпараметров для модели GradientBoostingClassifier
- код для скачивания таблицы с результатами
- код для скачивания модели

Файл app.py представляет из себя микро-приложение, созданное с помощью библиотеки Streamlit. Приложение позволяет вносить параметры онлайн и получать результат результат работы модели. Сервер запускается локально, поэтому для работы приложения необходимо скачать файл app.py и модель model.pcl и запустить в локальной среде.

Файл model.pcl - модель GradientBoostingClassifier
